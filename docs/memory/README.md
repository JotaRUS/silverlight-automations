# Project memory

This folder holds shared memory so we don’t repeat work and use fewer tokens on repeated instructions.

## Files

| File | Purpose |
|------|---------|
| [done_index.md](./done_index.md) | Mark today’s deliverable as completed. |
| [lessons_learned.md](./lessons_learned.md) | Add new learnings. |
| [error_patterns.md](./error_patterns.md) | Record confusions/mistakes and their fixes. |
| [open_questions.md](./open_questions.md) | Add unresolved items with concrete verification steps. |
| [backlog.md](./backlog.md) | Remove completed items and reprioritize next tasks. |

The agent is instructed (via Cursor rules) to update these at the end of tasks when relevant.
